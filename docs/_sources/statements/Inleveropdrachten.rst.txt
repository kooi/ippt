opg-statements-trap
-------------------

Teken de onderstaande figuur. Dit kun je doen door uitsluitend gebruik te maken van ``forward()`` en ``right()``.

.. image:: images/trap.png

.. activecode:: opg-statements-trap
   :caption: Trap
   :nocodelens:
   :language: python

   import turtle
   tina = turtle.Turtle()
   tina.shape("turtle")

   tina.forward(0)
   tina.right(0)
